Css基础 - 单选框与复选框伪类的应用

宗旨是在不需要用到 js脚本 的情况下实现“点击”操作 (使用伪类操作获取理想效果)

radio和checkbox的基本应用

input里的 id 与label里的 for属性 相互绑定

<div class="container">
        <!-- checked 默认选中 hidden 隐藏input -->
        <input type="radio" name="sex" id="btn" checked>
        <label for="btn">男</label>
        <input type="radio" name="sex" id="btn1">
        <label for="btn1">女</label>

        <input type="radio" name="sex" id="btn" hidden>
            
        <input type="checkbox" id="for_one"> 复选1
        <input type="checkbox" id="for_two"> 复选2
        <label for="for_one">我是文字复选 1</label>
        <label for="for_two">我是文字复选 2</label>
    </div>

根据上面基础加上:checkbox伪类 就能做到点击效果

:checkbox 理解及应用

:checked CSS 伪类选择器表示任何处于选中状态的radio(<input type="radio">), checkbox (<input type="checkbox">) 或(“select”) 元素中的option HTML元素(“option”)。

/* 匹配任意被勾选/选中的radio(单选按钮),checkbox(复选框),或者option(select中的一项) */
:checked {
  width: 50px;
  height: 50px;
  margin-left: 25px;
}

用户通过勾选/选中元素或取消勾选/取消选中，来改变该元素的 :checked 状态。

借用隐藏的checkbox来切换元素的样式（显示/隐藏)

这个例子利用了:checked伪类，让用户基于复选框的状态切换内容，而无需使用JavaScript。

<!-- 点击圆圈充满整个屏幕 -->  
  <div class="container">
      <input type="checkbox" name="" id="click_01" hidden>
      <label for="click_01" class="start_btn">s_Btn</label>
      <label for="click_01" class="end_btn">e_Btn</label>
      <div class="clickChange"></div>     
  </div>
<style>
        *{
            margin: 0;
            padding: 0;
        }
        .container{
            height: 100vh;
            display: flex;
            justify-content: center;
            align-items: center;
        }
        .clickChange{
           width: 150px;
           height: 150px;
           background-color: aqua;
           border-radius: 50%; 
           transition: 0.5s;
        }
        /* 开始按钮 */
        .start_btn{
            position: absolute;
            width: 50px;
            height: 50px;
            background-color: aquamarine;
            transition: 0.4s;
        }
        /* 结束按钮 */
        .end_btn{
            position: fixed;
            top: 10px;
            left: 10px;
            width: 50px;
            height: 50px;
            background-color: aquamarine;
            display: none;
        }
		/* ~ 通用兄弟选择器 */ 
        #click_01:checked ~ .clickChange{
           width: 100vw;
           height: 100vh;
           background-color: red;
           border-radius: 0; 
        }
        #click_01:checked ~ .start_btn{
            width: 20px;
            height: 20px;
            background-color: aquamarine;
            border-radius: 50%;
            display: none;
        }
        #click_01:checked ~ .end_btn{
            display: block;
        }
    </style>

进阶案例 - 全屏搜索栏

<div class="wrap">
        <input type="checkbox" id="search_btn" hidden>
        <label for="search_btn" class="search-btn">
            <i class="fa fa-search" aria-hidden="true"></i>
        </label>
        <input type="checkbox" id="search_btn" hidden>
        <label for="search_btn" class="close-btn">
            <i class="fa fa-close" aria-hidden="true"></i>
        </label>
        <div class="container">
            <div class="search-box">
                <input type="text" placeholder="请输入...">
                <i class="fa fa-search" aria-hidden="true"></i>
            </div>
        </div>
    </div>
*{
    margin: 0;
    padding: 0;
}
.wrap{
    height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}
.search-btn{
    position: relative;
    width: 60px;
    height: 60px;
    line-height: 60px;
    text-align: center;
    z-index: 1;
    cursor: pointer;
    /* display: none; */
}
.search-btn .fa{
    color: #fff;
    font-size: 22px;
}
/* 关闭按钮 */
.close-btn{
   display: none;
   position: absolute;
   top: 25px;
   right: 25px;
   font-size: 25px;
   color: #fff;
   cursor: pointer;
   z-index: 1;
}
.container{
   position: fixed;
   width: 100%;
   height: 100%;
   display: flex;
   justify-content: center;
   align-items: center;
   background: linear-gradient(200deg,#6e86ee,#453c90);
   clip-path: circle(30px at 50% 50%);
   /* 设置过度 */
   transition: 0.4s;
}
.search-box{
    /* display: none; */
    /* width: 400px; */
    /* 默认宽度为0 隐藏*/
    width: 0;
    height: 50px;
    display: flex;
    justify-content:space-between;
    border-bottom: 3px solid #fff;
    overflow: hidden;
    transition: 0.3s;
}
.search-box input{
    width: 100px;
    height: 50px;
    border: none;
    background: none;
    outline: none;
    color: #fff;
    font-size: 22px;
    text-indent: 8px;
}
.search-box input::placeholder{
    color:rgba(255,255,255,0.7)
}
.search-box .fa{
    width: 50px;
    height: 50px;
    line-height: 50px;
    text-align: center;
    color: #fff;
    font-size: 22px;
    cursor: pointer;
}

#search_btn:checked ~ .search-btn{
    display: none;
}
#search_btn:checked ~ .close-btn{
    display: block;
}
#search_btn:checked ~ .container{
    clip-path: circle(100%);
}
#search_btn:checked ~ .container .search-box{
    width: 400px;
}
