CSS画基本图形

有时候我的内心会问，这些图形还不如直接用Ps或者svg等一些工具完成。可能是打过之后细节都在其中。

1. 使用 border 属性达到一系列基本图形效果

主要运用的是 transparent值 达到透明的效果

<div class="wrap">
        <div class="circle">圆形</div>
        <div class="triangle">三角形</div>
        <div class="trapezium">等腰梯形</div>
        <div class="hexagon">
            <div class="top">六边形</div>
            <div class="bottom"></div>
        </div>
    </div>
.wrap{
          display: flex;
          flex-flow: row wrap;
          justify-content: space-between;
          align-content: space-between;
      }
      .circle{
          width: 100px;
          height: 100px;
          background: rgba(255, 5, 5, 0.979);
          border-radius: 50%;
      }
      .triangle{
          width: 0px;
          height: 0px;
         /*  background: rgba(255, 5, 5, 0.979); */
          border-top: 50px solid transparent;
          border-left: 50px solid  transparent;
          border-bottom: 50px solid pink;
          border-right: 50px solid transparent;
          /* transparent 为透明色 */
      }
      .trapezium{
          width: 50px;
          height: 0px;
          border-top: 50px solid transparent;
          border-left: 50px solid  transparent;
          border-bottom: 50px solid green;
          border-right: 50px solid transparent;
      }
      .hexagon .top{
          width: 50px;
          border-top: 50px solid transparent;
          border-left: 50px solid  transparent;
          border-bottom: 50px solid green;
          border-right: 50px solid transparent;
      }
      .hexagon .bottom{
          width: 50px;   
          border-top: 50px solid green;
          border-left: 50px solid  transparent;
          border-bottom: 50px solid transparent;
          border-right: 50px solid transparent;
      }

2. 使用 clip-path 属性 裁剪得到基本图形效果

简介

clip-path CSS 属性可以创建一个只有元素的部分区域可以显示的剪切区域。区域内的部分显示，区域外的隐藏。剪切区域是被引用内嵌的URL定义的路径或者外部svg的路径，或者作为一个形状。clip-path属性代替了现在已经弃用的剪切 clip属性。

兼容性

点击查看

基本语法

clip-source | basic-shape | geometry-box | none

属性说明

clip-source = url
basic-shape = inset | circle | ellipse | polygon
geometry-box = shape-box | fill-box | stroke-box | view-box

语法详解

基本图形：inset

inset() : 定义一个矩形 。注意，定义矩形不是rect，而是 **inset**。

//语法
inset( <length-percentage>{1,4} [ round <border-radius> ]? )
//说明
inset()可以传入5个参数，分别对应top,right,bottom,left的裁剪位置,round radius（可选，圆角）

//示例
clip-path: inset(2em 3em 2em 1em round 2em);
基本图形：circle

circle() : 定义一个圆 。

//语法
circle( [ <shape-radius> ]? [ at <position> ]? )
//说明
circle()可以传人2个可选参数；
1. 圆的半径，默认元素宽高中短的那个为直径，支持百分比
2. 圆心位置，默认为元素中心点
//示例
clip-path: circle(30% at 150px 120px);
基本图形：ellipse

ellipse() : 定义一个椭圆 。

//语法
ellipse( [ <shape-radius>{2} ]? [ at <position> ]? )
//说明
ellipse()可以传人3个可选参数；
1. 椭圆的X轴半径，默认是宽度的一半，支持百分比
2. 椭圆的Y轴半径，默认是高度的一半，支持百分比
3. 椭圆中心位置，默认是元素的中心点

//示例
clip-path: ellipse(45% 30% at 50% 50%);
基本图形：polygon

polygon() : 定义一个多边形 。

//语法
polygon( <fill-rule>? , [ <length-percentage> <length-percentage> ]# )
//说明
<fill-rule>可选，表示填充规则用来确定该多边形的内部。可能的值有nonzero和evenodd,默认值是nonzero
后面的每对参数表示多边形的顶点坐标（X,Y），也就是连接点

//示例
clip-path: polygon(50% 0,100% 50%,0 100%);

使用实例

<!-- clip-path属性的应用  -->
     <!-- clip-path CSS 属性使用裁剪方式创建元素的可显示区域。区域内的部分显示，区域外的隐藏。-->
   <!-- 可以用table表格布局不过，懒直接flex算了 -->
   <div class="wrap">
       <div class="circle">圆形</div>
       <div class="ellipse">椭圆</div>
       <div class="rectangle">矩形</div>
       <div class="triangle">三角形</div>
       <div class="rhombus">菱形</div>
       <div class="trapezoidal">梯形</div>
       <div class="parallelogram">平行四边形</div>
       <div class="pentagon">五边形</div>
       <div class="hexagon">六边形</div>
       <div class="heptagon">七边形</div>
       <div class="octagon">八边形</div>
       <div class="obliqueAngle">斜角</div>
       <div class="notch">槽口</div>
       <div class="leftArrow">左箭头</div>
       <div class="star">星星</div>
       <div class="crosses">十字架</div>
       <div class="fork">叉号</div>
       <div class="dialogBox">对话框</div>

       <div class="deformation">变形效果</div>
   </div>
.wrap{
            display: flex;
            flex-flow: row wrap;
            justify-content: space-between;
            align-content: space-between;
        }
        .wrap div{
            width: 100px;
            height: 100px;
            background-color: orange;
        }
        /* 圆形裁剪 */
        .circle{
            clip-path: circle(50% at 50% 50%);
        }
        /* 椭圆裁剪 */
        .ellipse{
            clip-path: ellipse(25% 40% at 50% 50%);
        }
        /* 矩形裁剪 */
        .rectangle{
            clip-path: inset(5% 20% 15% 10%);
        }
        /* 多边形裁剪 - 三角形 */
        .triangle{
            clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
        }
        /* 多边形裁剪 - 菱形 */
        .rhombus{
            clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
        }
        /* 多边形裁剪 - 梯形 */
        .trapezoidal{
            clip-path: polygon(25% 0%,75% 0%,100% 100%,0% 100%);
        }
        /* 多边形裁剪 - 平行四边形 */
        .parallelogram{
            clip-path: polygon(25% 0%, 100% 0%, 75% 100%, 0% 100%);
        }
        /* 多边形裁剪 - 五边形 */
        .pentagon{
            clip-path: polygon(50% 0%, 100% 38%, 82% 100%, 18% 100%, 0% 38%);
        }
        /* 多边形裁剪 - 六边形 */
        .hexagon{
            clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0% 75%, 0% 25%);
        }
        /* 多边形裁剪 - 七边形 */
        .heptagon{
            clip-path: polygon(50% 0%, 90% 20%, 100% 60%, 75% 100%, 25% 100%, 0% 60%, 10% 20%);
        }
        /* 多边形裁剪 - 八边形 */
        .octagon{
            clip-path: polygon(30% 0%, 70% 0%, 100% 30%, 100% 70%, 70% 100%, 30% 100%, 0% 70%, 0% 30%);              
        }
        /* 多边形裁剪 - 斜角 */
        .obliqueAngle{
            clip-path: polygon(20% 0%, 80% 0%, 100% 20%, 100% 80%, 80% 100%, 20% 100%, 0% 80%, 0% 20%);
        }
        /* 多边形裁剪 - 槽口 */
        .notch{
            clip-path: polygon(0% 15%, 15% 15%, 15% 0%, 85% 0%, 85% 15%, 100% 15%, 100% 85%, 85% 85%, 85% 100%, 15% 100%, 15% 85%, 0% 85%);
        }
        /* 多边形裁剪 - 左箭头 */
        .leftArrow{
            clip-path: polygon(40% 0%, 40% 20%, 100% 20%, 100% 80%, 40% 80%, 40% 100%, 0% 50%);
        }
        /* 多边形裁剪 - 星星 */
        .star{
            clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%);
        }
        /* 多边形裁剪 - 十字架 */
        .crosses{
            clip-path: polygon(10% 25%, 35% 25%, 35% 0%, 65% 0%, 65% 25%, 90% 25%, 90% 50%, 65% 50%, 65% 100%, 35% 100%, 35% 50%, 10% 50%);
        }
        /* 多边形裁剪 - 叉号 */
        .fork{
            clip-path: polygon(20% 0%, 0% 20%, 30% 50%, 0% 80%, 20% 100%, 50% 70%, 80% 100%, 100% 80%, 70% 50%, 100% 20%, 80% 0%, 50% 30%);
        }
        /* 多边形裁剪 - 对话框 */
        .dialogBox{
            clip-path: polygon(0% 0%, 100% 0%, 100% 75%, 75% 75%, 75% 100%, 50% 75%, 0% 75%);               
        }
        /* 变形效果 */       
        .deformation{
            clip-path: polygon(20% 0%, 80% 0%, 100% 20%, 100% 80%, 80% 100%, 20% 100%, 0% 80%, 0% 20%);
            transition: 0.5s;
        }
        .deformation:hover{
            clip-path:polygon(0 0,0 0,100% 0,100% 0,100% 100%,100% 100%,0 100%,0 100%);
        }
